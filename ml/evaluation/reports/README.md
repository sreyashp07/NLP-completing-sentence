# Evaluation Reports

Auto-generated by running:

```bash
py ml/evaluation/evaluator.py
```

## Files generated

- confusion_matrix.png — Heatmap of predicted vs true labels
- per_class_metrics.png — Precision, Recall, F1 bar charts per class
- metrics_report.csv — Per-class metrics in tabular format
- summary.yaml — Overall metrics summary

## Regenerate

Run evaluator any time after retraining to refresh these reports.
