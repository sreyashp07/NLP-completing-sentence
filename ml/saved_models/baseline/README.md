# Baseline Model Artifacts

These files are generated by running the training script.

## Generate

```bash
python data/generate_dataset.py
python ml/training/train_baseline.py
```

## Files

- pipeline.pkl — TF-IDF + Logistic Regression sklearn pipeline
- label_encoder.pkl — LabelEncoder mapping intent names to integers
- metrics.yaml — Training metrics (accuracy, F1 weighted/macro)
- classes.txt — List of 9 intent class names

## Model Info

- Algorithm: TF-IDF + Logistic Regression
- Features: 15,000 TF-IDF features (1-3 ngrams)
- Classes: 9 intent categories
- Training samples: 1440
- Test samples: 360
- Accuracy: 1.0 on synthetic data, ~88% on real data
